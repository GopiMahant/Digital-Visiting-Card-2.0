# CardForge — Your Digital Identity, Crafted

A premium digital business card creator with 10 unique templates, live preview, and export options.

## Features

- **10 Premium Templates**: From Obsidian Executive to Carbon Brutalist
- **Live Preview**: Real-time updates as you type
- **Logo Upload**: Custom logos with template fallbacks
- **Export Options**: PNG and PDF downloads
- **Responsive Design**: Works on desktop and mobile

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser or serve with a local server
3. Create your card!

## Project Structure

```
cardforge/
├── index.html          # Main app
├── style.css           # Global styles
├── app.js              # Main controller
├── templates/          # Template definitions
├── components/         # Component modules
└── assets/
    └── default-logos/  # Template logos
```

## Technologies

- HTML5, CSS3, JavaScript
- html2canvas for PNG export
- jsPDF for PDF export
- Google Fonts for typography
